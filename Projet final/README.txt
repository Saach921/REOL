MODE D'EMPLOI 

1) Créer l'environnement conda par la commande : conda env create -f reol.yml 

2) Activer le avec : conda activate env_reol

3) Vous pouvez alors lancer le programme avec : python reol_f.py --lat=[Entrez la latitude souhaitée] --long=[Entrez la longitude souhaitée]  --model=[Entrez le modèle "squeezenzt" ou "efficientnet"]

/!\ Les latitude et longitude doivent être entrées en degré décimal - par exemple --lat=48.623 --long=4.450 /!\
